# openpose-leg 使用指南

## 📌 项目介绍

本项目基于 [CMU OpenPose](https://github.com/CMU-Perceptual-Computing-Lab/openpose) 改造，**专注腿部姿态估计**，识别以下 6 个关键点：

| 编号 | 部位 | 说明 |
|------|------|------|
| 8 | 右髋 | Right Hip |
| 9 | 右膝 | Right Knee |
| 10 | 右踝 | Right Ankle |
| 11 | 左髋 | Left Hip |
| 12 | 左膝 | Left Knee |
| 13 | 左踝 | Left Ankle |

骨骼连接关系：髋 → 膝 → 踝（左右各一条）

---

## 🚀 快速开始

### 第一步：安装 Anaconda

前往 [Anaconda 官网](https://www.anaconda.com/download) 下载安装包，一路默认安装即可。

> 💡 如果不想用 Anaconda，Python 3.7+ + pip 也可以。

### 第二步：创建虚拟环境

打开终端（或 Anaconda Prompt），执行：

```bash
conda create -n pytorch python=3.7 -y
conda activate pytorch
```

### 第三步：安装 PyTorch

- **有 NVIDIA 显卡（推荐）**：前往 [PyTorch 官网](https://pytorch.org/get-started/locally/) 选择你的 CUDA 版本，复制安装命令执行
- **没有显卡**：同样去官网选 CPU 版本

### 第四步：安装其他依赖

```bash
pip install -r requirements.txt
```

如果个别包安装失败，可以用国内镜像加速：

```bash
pip install opencv-python -i https://pypi.tuna.tsinghua.edu.cn/simple
pip install matplotlib -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 第五步：下载模型

模型文件 `body_pose_model.pth`（199MB）太大无法上传到 GitHub，请从网盘下载：

> **百度网盘**：[https://pan.baidu.com/s/1r47RZDqcIdJScpJBXfv2cw](https://pan.baidu.com/s/1r47RZDqcIdJScpJBXfv2cw)
> **提取码**：`t8qi`

下载后，将 `body_pose_model.pth` 放入项目根目录的 `model/` 文件夹：

```
openpose-leg/
└── model/
    └── body_pose_model.pth   ← 放这里
```

## 🎮 使用方法

### 1. 摄像头实时检测

```bash
python demo_camera.py
```

直接打开摄像头，实时显示腿部骨架。按 `q` 退出。

### 2. 图片检测

编辑 `demo.py`，修改第 12 行的图片路径：

```python
test_image = 'images/ski.jpg'  # 改成你的图片路径
```

然后运行：

```bash
python demo.py
```

结果会保存为 `images/output.png`。

### 3. 视频检测

```bash
python demo_video.py --file 你的视频.mp4
```

> ⚠️ **必须在终端运行**，不要在 IDE 里点"运行"。

输出文件会在原视频同目录下生成 `*.processed.mp4`。

**常用参数：**

```bash
# 全帧处理（最清晰，但最慢）
python demo_video.py --file video.mp4 --skip 1

# 缩小画面提速
python demo_video.py --file video.mp4 --scale 0.5

# 降低模型精度换速度
python demo_video.py --file video.mp4 --boxsize 256

# 关闭预览窗口（可提速）
python demo_video.py --file video.mp4 --no-preview

# 组合使用
python demo_video.py --file video.mp4 --skip 1 --scale 0.5 --no-preview
```

### 4. Web 界面

```bash
python web/app.py
```

浏览器打开 `http://localhost:5000`，上传图片或视频即可处理。

## 💡 常见问题

### Q: 没有显卡能用吗？
可以，但会比较慢。CPU 模式下建议加上 `--scale 0.5 --boxsize 256 --skip 3`。

### Q: GPU 利用率不高怎么办？
确保安装了 CUDA 版的 PyTorch（不是 CPU 版）。项目已启用 FP16 半精度推理，GPU 利用率应该较高。如果仍然低，可能是 `--boxsize` 太小或视频分辨率太低。

### Q: 视频处理太慢怎么办？
- 加 `--scale 0.5`（最有效的提速方式）
- 加 `--skip 2`（跳帧，处理一半帧数）
- 加 `--no-preview`（关闭预览窗口）
- 加 `--boxsize 256`（降低模型精度）

### Q: opencv 安装报错？
```bash
pip install opencv-python -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q: 想打印关键点坐标？
在 `src/util.py` 的 `draw_bodypose` 函数中，找到注释掉的 `print(i, x, y)`，取消注释即可。关键点编号：8(右髋), 9(右膝), 10(右踝), 11(左髋), 12(左膝), 13(左踝)。
